# Проект "Закрывающий тег"

В данном проекте реализована вёрстка одностраничного сайта для практики знаний по SVG, анимациям, вариативным шрифтам и декорированию по макету из [Figma](https://clck.ru/3SvVCy).

## Использование
Установите npm-пакет перед началом с помощью команды:

```sh
$ npm i
```
